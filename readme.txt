MotorSqlite is an aiosqlite (https://pypi.org/project/aiosqlite/) wrapper
to fit the motor (https://github.com/mongodb/motor/) api interface for the 
repo (https://github.com/PloxHost-LLC/community-bot) PR #88 for async sqlite
support without changing the source code.

This code is posted in this repository in the hopes it is useful to someone,
in any shape or form.
